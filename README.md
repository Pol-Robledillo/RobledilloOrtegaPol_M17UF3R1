# RobledilloOrtegaPol_M17UF3R1
## **Game Controls**
- `W` / `A` / `S` / `D` to move the character
- `Space` to jump
- `Shift` to crouch
- `Ctrl` to sprint
- `Left Click` to shoot
- `Right Click` to aim
- `1` / `2` / `3` to change weapons (you need to pick up the 2 and 3 weapons to use them)
- `F1` to change to first person
- `F2` to change to third person
- `X` to dance
